06 / Usage Guidelines
Brand Rules
Consistent application ensures the Algora brand remains cohesive across all touchpoints, from the web platform to social media and print materials.
Do
Maintain clear space around the logo. Use it on dark backgrounds. Keep golden amber for primary CTAs only. Use Inter for text and IBM Plex Mono for code. Maintain RTL support for Arabic content.
Do
Keep body text at 14-16px for readability. Use semantic color meanings consistently. Test all pages in both RTL and LTR layouts. Ensure minimum contrast ratio of 4.5:1 for body text.
Don't
Never stretch, rotate, or distort the logo. Never place it on busy backgrounds. Never use golden accent for more than one primary CTA per screen. Never use light mode as the default.
Don't
Never use more than 5 distinct colors per page. Never mix Google Translate with hand-crafted content. Never use decorative borders on body pages. Never add stock images or clipart for decoration.
