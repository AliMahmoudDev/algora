01 / Brand Overview
What is Algora?
Algora is a bilingual algorithms and problem-solving education platform built for developers worldwide. The name combines "Algorithm" with "Ora" (light in Latin), symbolizing enlightenment through code. Algora empowers developers to master computational thinking with an AI-powered learning assistant.

Brand Personality
Algora is intelligent yet approachable, technical yet human. It represents the intersection of deep computer science knowledge and modern AI-assisted learning. The brand speaks with clarity and precision, never overwhelming, always guiding.

03 / Logo System
Primary Logo
The Algora logo combines geometric code syntax with an algorithmic tree structure, representing the fusion of programming fundamentals and computational thinking.
ALGORA
Master Algorithms. Code the Future.
Logo Concepts
Concept A
Code Bracket
Abstract < > symbols forming a stylized A. Represents code syntax as the foundation of all programming.
Concept B
Algorithm Tree
Binary tree node structure with golden connections. Represents data structures and hierarchical thinking.
Concept C
Syntax A
Letter A integrated with code syntax elements. Represents the fusion of alphabet and algorithm.
Logo Clear Space
Always maintain minimum clear space around the logo equal to the height of the letter "A" in the logotype. Never place the logo on busy or patterned backgrounds without a solid container. The minimum reproduction size is 24px height for digital and 8mm for print.

06 / Usage Guidelines
Brand Rules
Consistent application ensures the Algora brand remains cohesive across all touchpoints, from the web platform to social media and print materials.
Do
Maintain clear space around the logo. Use it on dark backgrounds. Keep golden amber for primary CTAs only. Use Inter for text and IBM Plex Mono for code. Maintain RTL support for Arabic content.
Do
Keep body text at 14-16px for readability. Use semantic color meanings consistently. Test all pages in both RTL and LTR layouts. Ensure minimum contrast ratio of 4.5:1 for body text.
Don't
Never stretch, rotate, or distort the logo. Never place it on busy backgrounds. Never use golden accent for more than one primary CTA per screen. Never use light mode as the default.
Don't
Never use more than 5 distinct colors per page. Never mix Google Translate with hand-crafted content. Never use decorative borders on body pages. Never add stock images or clipart for decoration.
